Read It Aloud — how every lesson works

Every formula is broken down argument by argument into plain language. Once you can say it out loud — in any language — you own it.

=VLOOKUP(A2, products!A:D, 3, FALSE)
A2→the value we're searching for
products!A:D→search in this table (first column gets searched)
3→return the value from the 3rd column of that range
FALSE→exact match only — no close guesses
